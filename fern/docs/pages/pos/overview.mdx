---
title: "Introduction to POS API"

---

The POS APIs are meant to allow a wide range of use cases, including:
- Custom UI for themed unattended POS that only uses the backend for business logic.
- Custom UI for QR selfservice POS apps that only uses the backend for business logic.
- Custom UI for customer facing displays that show newly added lines in real time.
- Looking up finished sales, known as POS entries.
- Looking up and creating new POS Units.
