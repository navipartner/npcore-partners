---
title: Get Server Instance ID
description: Call this endpoint to get the node server instance ID the API call is processed on.
---

This endpoint is used by NaviPartner services to get the server instance ID the API call is processed on.
It can be used to identify what server instance is linked with the Entra App used for the API call. 
Server instances for one Entra App can fluctuate over time and this endpoint will provide the most recent server instance ID.
